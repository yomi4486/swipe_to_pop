# SwipeToPopApp
スワイプで画像を仕分けできるアプリ

## 概要
このアプリは、マッチングアプリのような直感的なスワイプ操作で画像を効率的に仕分けできるiOSアプリです。写真ライブラリの大量の画像を素早く整理したい方に最適です。

## 機能
- **スワイプ操作**: 左右のスワイプで画像に対してアクションを実行
- **カスタマイズ可能なアクション**: 左右スワイプに異なるアクションを設定可能
- **4つのアクション**:
  - 🗑️ **削除**: 画像を完全に削除
  - 👁️‍🗨️ **非表示**: 画像を一時的に非表示
  - 📁 **フォルダ移動**: 特定のフォルダに移動
  - ❤️ **お気に入り**: 画像をお気に入りに追加

## システム要件
- iOS 15.0以降
- Xcode 14.0以降
- Swift 5.8以降

## セットアップ方法

### 1. プロジェクトのクローン
```bash
git clone https://github.com/yomi4486/swipe_to_pop.git
cd swipe_to_pop
```

### 2. Xcodeでプロジェクトを開く
```bash
open SwipeToPopApp.xcodeproj
```

または、Swift Package Managerを使用する場合:
```bash
swift package generate-xcodeproj
open SwipeToPopApp.xcodeproj
```

### 3. 実機またはシミュレータでビルド・実行

## 使用方法

### 基本操作
1. アプリを起動すると、写真ライブラリへのアクセス許可を求められます
2. 許可すると、最新の画像から順番に表示されます
3. 画像を左右にスワイプして仕分けします:
   - **左スワイプ**: デフォルトで削除（設定変更可能）
   - **右スワイプ**: デフォルトでお気に入り（設定変更可能）

### 設定変更
1. 右上の歯車アイコンをタップして設定画面を開きます
2. 左スワイプ・右スワイプのアクションを変更できます
3. 統計情報も確認できます

## 技術詳細

### アーキテクチャ
- **SwiftUI**: モダンなUI構築フレームワーク
- **MVVM**: Model-View-ViewModelアーキテクチャパターン
- **Photos Framework**: 写真ライブラリへのアクセス
- **ObservableObject**: リアクティブなデータバインディング

### 主要コンポーネント

#### ImageSortingModel
- 写真ライブラリからの画像読み込み
- スワイプアクションの実行
- アプリの状態管理

#### ContentView
- メイン画面のUI
- ドラッグジェスチャーの処理
- 画像の表示とアニメーション

#### SettingsView
- スワイプアクションの設定
- 統計情報の表示
- アプリの説明

## パーミッション
このアプリは以下のパーミッションが必要です:
- **NSPhotoLibraryUsageDescription**: 画像の表示・整理
- **NSPhotoLibraryAddUsageDescription**: お気に入り機能
